== Description ==
Wp TrovaPrezzi Wordpress / Woocommerce Plugin, è un plugin Wordpress che permette di entrare a far parte del mondo TrovaPrezzi.it in maniera semplice e veloce. Funziona con WooCommerce. ( Trova Prezzi )
Con Wp TrovaPrezzi Wordpress / Woocommerce Plugin, ti basterà scegliere quali prodotti esportare, ed in un semplice click le tue offerte saranno su TrovaPrezzi.it

== Installation ==
Installazione e Configurazione:

1. Dopo aver installato ed attivato il plugin, vai in Impostazioni -> TrovaPrezzi e segui le istruzioni a schermo.
2. Il tab IMPOSTAZIONI consente la creazione del CSV che verrà creato.
3. Il tab PRODOTTI SU TROVAPREZZI consente di verificare quali prodotti sono stati selezionati per essere esportati su TrovaPrezzi e di verificare che i campi obbligatori siano tutti presenti.
4. Per scaricare il CSV, dal tab IMPOSTAZIONI in basso, clicca su \"DOWNLOAD CSV\".
